### Installation

1. Clone the repo

   ```sh
   git clone git@github.com:umairfarooq44/kosmic-challenge.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Run the Project
   ```sh
    npm run dev
   ```
   Project is now running at `http://localhost:3000/`
